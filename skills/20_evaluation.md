# 講評與總評 (Evaluation)

## 使用定位
- 本 skill 用於協助講評者、總評者、主席或教練提供具體、正向、可執行的回饋。
- 講評不是評斷人格，也不是只挑錯；目標是幫助講員知道「哪裡有效、哪裡可調、下一次怎麼做」。
- 若涉及社團正式評分規則、上台規範或流程，需優先依據 `knowledge/90_club_manual.md`。

## 講評前先確認
- 講員程度：新手、一般社員、進階講員、比賽講員。
- 演講目標：破冰、說服、故事、教學、幽默、社務說明、比賽稿。
- 時間限制：三分鐘講評時，不要塞太多點。
- 講員需求：若知道講員想練聲音、結構或肢體，優先回應該需求。
- 觀察素材：若只有文字稿，優先評內容與結構；若有現場觀察，才能評聲音、眼神、走位與觀眾反應。

## 講評框架選擇
- 新手或第一次上台：用 CRC 或「保留 / 微調 / 下一步」，先保護上台意願。
- 一般講員：用「亮點 / 盲點 / 練習點」，兼顧肯定與改進。
- 內容不清楚：用「主旨 / 結構 / 證據」檢查內功。
- 表現削弱內容：用「聲音 / 眼神 / 身體 / 節奏」檢查外功。
- 比賽稿或重要演講：用「訊息 / 張力 / 記憶點 / 舞台」做高標準講評。
- 總評全場：用「流程 / 角色 / 氣氛 / 學習點」整理，不逐字重評每位講員。

## CRC 講評架構

### C：Commend 肯定有效處
- 指出具體行為，而不是只說「很棒」。
- 說明這個行為對聽眾造成什麼效果。
- 例：「你在第二段放慢速度，讓我更容易感受到那個轉折。」

### R：Recommend 提出建議
- 一次以 1 到 2 個建議為主，避免讓講員無法吸收。
- 建議要可執行，可以在下一次演講直接練。
- 用「如果下一次...效果會更好」取代單純否定。

### C：Commend / Challenge 收束
- 回到講員優勢，讓講員願意繼續練。
- 可加一個具體挑戰，例如「下次試著把結尾停頓拉長兩秒」。

## 三段式講評模板
1. 我看到的核心特色：先描述講員這次最明顯的風格或主軸。
2. 最有效的地方：挑 1 到 2 個具體優點，說明為什麼有效。
3. 下一次可調整：挑 1 個最高槓桿建議，附上可操作方法。

## 常用講評方法庫

### 保留 / 微調 / 下一步
- 保留：指出最值得留下的具體做法。
- 微調：提出一個小到下次能做到的調整。
- 下一步：給一個明確練習任務。
- 適合：新手、時間很短、講員容易緊張時。

### 亮點 / 盲點 / 練習點
- 亮點：講員已經做對且可延續的地方。
- 盲點：講員可能沒注意到、但影響聽眾接收的地方。
- 練習點：下一次最值得練的單一技能。
- 適合：一般講員、社課講評、教練式回饋。

### 目標 / 現況 / 差距 / 行動
- 目標：這篇演講原本想達成什麼效果。
- 現況：現場實際呈現的效果。
- 差距：兩者之間最大的落差。
- 行動：下一次如何縮小落差。
- 適合：使用者有明確演講目的，例如說服、教學、比賽。

### 內容 / 結構 / 表達
- 內容：主旨、例子、觀點是否成立。
- 結構：開場、段落、轉場、結尾是否形成推進。
- 表達：聲音、眼神、手勢、節奏是否服務內容。
- 適合：使用者貼講稿或要求完整修改。

### 聽眾旅程法
- 一開始：聽眾有沒有被帶進題目。
- 中段：聽眾是否跟得上故事或論點。
- 轉折：聽眾是否感受到變化、驚喜或領悟。
- 結尾：聽眾是否帶走清楚訊息。
- 適合：故事演講、情緒型演講、成果發表。

### 高低槓桿法
- 高槓桿：一改就能明顯提升效果的問題，優先講。
- 低槓桿：修了也不太影響整體的細節，少講或不講。
- 取捨：三分鐘講評最多處理一個高槓桿建議。
- 適合：講員問題很多、講評時間很短時。

### 證據式講評
- 觀察：先講具體行為，不先貼標籤。
- 影響：說明這個行為對聽眾造成什麼效果。
- 建議：提出可操作的替代做法。
- 句型：「我觀察到你在 X 時做了 Y，這讓聽眾 Z；下次可以試 A。」
- 適合：避免講評流於主觀喜好。

### 比賽稿四問
- 訊息：聽眾聽完會記得哪一句話？
- 張力：故事裡有沒有清楚衝突、選擇或代價？
- 記憶點：有沒有畫面、金句、動作或反覆意象？
- 舞台：聲音、停頓、走位是否放大情緒？
- 適合：比賽、成果發表、重要演講精修。

### 總評四層
- 流程：時間、順序、規則說明是否清楚。
- 角色：主席、講員、講評、計時與場務是否各司其職。
- 氣氛：現場是否支持學習、回饋是否正向具體。
- 學習點：下次最值得保留與改善的是什麼。
- 適合：總評全場與課後檢討。

## 講評建議排序
1. 先處理會讓聽眾聽不懂的問題：主旨不清、結構混亂、例子無關。
2. 再處理會削弱效果的問題：音量、語速、停頓、眼神、手勢。
3. 最後才處理美感細節：字句精煉、金句、幽默、舞台調度。
4. 若講員是新手，即使內容問題不少，也先給一個能建立信心的練習點。
5. 若講員是進階者，可以提高標準，但仍要提供具體替代方案。

## 內功 / 外功講評法

### 評什麼
- 外功：看得到、聽得到的表現技巧，例如聲音、語速、停頓、眼神、手勢、站姿、移動、表情、台風、道具或投影片使用。
- 內功：支撐演講的思考與結構，例如破題、主旨、角度、邏輯、故事選材、段落安排、轉場、情緒鋪陳、說服策略、結尾回扣。
- 內外功要分開看，但不要割裂；外功服務內功，內功決定外功要怎麼用。

### 先評內功的情境
- 題目沒有破清楚，聽眾不知道這篇到底要回答什麼。
- 主旨太散，三分鐘內同時講太多觀點。
- 故事很好，但和題目或主張連不起來。
- 結構跳躍，聽眾跟不上段落順序。
- 結尾沒有回扣開場或破題。

### 先評外功的情境
- 講員是新手，最需要先建立上台安全感與基本台風。
- 內容已經清楚，但聲音、眼神、停頓或手勢削弱效果。
- 講員緊張導致語速過快、音量不足或小動作太多。
- 這次訓練目標本來就是聲音、肢體、時間或舞台呈現。

### 內功講評問題
- 破題是否清楚？有沒有指出題目的核心矛盾或角度？
- 主旨是否能用一句話說明？
- 每個故事或例子是否都支撐主旨？
- 段落順序是否形成推進，而不是資料堆疊？
- 結尾是否回答了開場提出的問題？

### 外功講評問題
- 聲音是否讓最後一排聽得清楚？
- 速度是否讓聽眾有時間吸收？
- 停頓是否放在重點前後？
- 眼神是否看向具體聽眾，而不是飄走？
- 手勢是否放大意思，而不是干擾內容？
- 站姿與移動是否配合段落轉換？

### 內外功整合句型
- 「你的內功很清楚，主旨是 X；如果外功上把 Y 放慢，這個主旨會更有力量。」
- 「你的外功很穩，台風讓人願意聽；下一步可以補強內功，把破題從 A 收斂成 B。」
- 「這篇最可惜的不是聲音，而是主旨太多；先把內功收斂，外功才知道要強調哪一句。」
- 「這次我會優先建議你練外功，因為內容已經成立，只要停頓和眼神更穩，說服力會上來。」

## 講評時間分配

### 三分鐘講評標準版
1. 0:00-0:20 開場定位：一句話說出講員這次的核心特色或你看到的主軸。
2. 0:20-1:05 肯定一個內功或外功亮點：說具體行為與效果。
3. 1:05-2:10 提出一個最高優先建議：說問題、原因、改法。
4. 2:10-2:40 示範或替代句：給講員下一次能直接照做的版本。
5. 2:40-3:00 收束鼓勵：回到講員優勢，給一個明確練習方向。

### 新手講員時間分配
- 40% 肯定：先保護上台意願，肯定勇氣與已做到的具體行為。
- 40% 一個外功建議：例如音量、眼神、站穩、放慢。
- 20% 收束：給一句簡單可練的提醒。

### 進階講員時間分配
- 25% 肯定：指出高階有效處，不空泛稱讚。
- 50% 內功分析：破題、主旨、結構、情緒、說服策略。
- 15% 外功微調：停頓、眼神、走位、重音。
- 10% 挑戰：給下一次更高標準的任務。

### 比賽稿或重要演講時間分配
- 20% 核心效果：這篇最打動人的地方是什麼。
- 45% 內功取捨：主旨、故事張力、笑點、轉折、金句、結尾。
- 25% 外功精修：節奏、停頓、舞台調度、表情、重音。
- 10% 記憶點：最後留下哪一句或哪個畫面。

### 總評時間分配
- 若是總評全場，可用「流程、角色、氣氛、下次建議」四段整理。
- 建議比例：流程 25%、角色 30%、氣氛 20%、下次建議 25%。
- 總評不應重講個別講員的完整講評，只補充個別講評沒有照顧到的共通問題。
- 總評也可分內外功：內功看課程設計、角色分工、學習目標；外功看時間、場務、轉場、音量、秩序與氣氛。

## 觀察面向

### 內容
- 主旨是否清楚？
- 開場是否抓住注意力？
- 論點是否有例子支撐？
- 結尾是否回扣主旨？
- 有沒有離題或資訊太多？

### 結構
- 段落順序是否自然？
- 轉場是否清楚？
- 是否有明確的起、承、轉、合？
- 三分鐘演講是否只保留一個核心訊息？

### 聲音
- 音量是否足夠？
- 速度是否穩定，重點是否有放慢？
- 語調是否有起伏？
- 停頓是否幫助聽眾吸收？

### 肢體與眼神
- 眼神是否看向具體聽眾？
- 手勢是否配合內容？
- 站姿是否穩定？
- 移動是否有目的？
- 是否有干擾性小動作？

### 舞台效果
- 情緒是否真誠？
- 幽默是否服務主題？
- 聽眾是否能跟上？
- 講員的特色是否被放大？

## 給新手的講評方式
- 優先保護上台意願，先肯定勇氣與已做到的具體行為。
- 建議只給一個，且要小到下一次能做到。
- 避免一開始就談太多高階技巧。
- 可用句型：「下一次你只要先練一件事，就是...」

## 給進階講員的講評方式
- 可以提高標準，但仍需具體。
- 聚焦在訊息精準度、段落取捨、情緒層次、舞台調度與觀眾反應。
- 可提出替代句、替代轉場或重新排序方式。
- 若是比賽稿，要檢查主旨、故事張力、笑點位置、金句、時間與記憶點。

## 總評者觀察清單
- 流程是否準時開始、準時結束？
- 主席是否清楚交代規則、角色與轉場？
- 講員、講評、計時、場務是否各司其職？
- 整體氣氛是否支持學習？
- 個別講評是否具體、正向、可執行？
- 下次課程最值得保留與最該改善的地方是什麼？

## 講評禁忌
- 不做人身評價，例如「你就是不適合...」。
- 不把個人喜好包裝成絕對規則。
- 不一次塞五六個缺點。
- 不只講「很好」卻沒有原因。
- 不在總結時新增新的批評點。
- 不替講員重寫整篇，除非使用者明確要求。

## 回答格式建議
- 使用者問「怎麼講評」：給一個三段式模板。
- 使用者貼演講內容：先講內容與結構，再講聲音肢體的假設性建議。
- 使用者貼實際觀察：根據觀察給具體講評稿。
- 使用者要總評：用「流程、角色、氣氛、下次建議」四段整理。
