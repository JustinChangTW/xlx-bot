# 台北市健言社 (Taipei Toastmasters Club) 核心知識庫

## 🟢 社團宗旨與核心精神
- **社團宗旨**：提供社會青年進修機會，同登學習、服務、領導領域。
- **健言信條**：認同本社宗旨，接受訓練，進而服務社會。
- **精神口號**：「一朝健言，終身健言。接受訓練，服務社會。」
- **核心訓練指標**：公眾演講、領導才能、聆聽溝通、人際關係、判斷包容、擔當回饋。

## 🔵 社課與活動運作 (第 159 期指標)
- **活動時間**：每週四 19:00 - 21:40
- **活動地點**：延平區民活動中心 (台北市大同區保安街47-1號3樓)
- **標準社課流程**：
  - 19:00 - 19:25：熱力學堂 / TED 時間 (暖身動腦，卸下疲累)
  - 19:25 - 20:30：TM (Toastmasters) 訓練 (每週特定主題演講與實作)
  - 20:30 - 20:40：中場休息與社務報告
  - 20:40 - 21:40：教育訓練 (專業講師授課)
- **第 159 期教育訓練主題參考**：
  - 破冰建立：口才破冰術 (克服上台緊張、破冰金句)
  - 講稿撰寫：撰稿順口溜 (文字轉化為口語邏輯、演講黃金結構)
  - 互動技巧：學會問對問題 (封閉/開放式提問、引導對話、Q&A 應變)
  - 舞台魅力：演講風格賞析 (風格辨識、找尋專屬表達頻率)
  - 評估回饋：講評萬萬歲 (回饋藝術、同理心觀察)

## 🟡 TM 訓練體系與口語表達技巧
- **三明治講評法 / CRC 規則 (核心講評架構)**：
  - **C (Commend - 讚美)**：用肯定與讚美開頭，建立講員自信，打造正向交流氛圍。
  - **R (Recommend - 建議)**：精準抓出重點，短時間拆解優劣，給予具體且具邏輯的改善建議。
  - **C (Commend - 鼓勵)**：用溫暖的鼓勵結尾，讓回饋成為推動他人進步的力量，讓人「心服口服」。
- **計時與規範**：
  - 確保每位講員的發言在規定時間內完成。
  - 鈴聲規範（例：三分鐘按鈴一次，三分半按鈴兩次，四分鐘強迫下台）。
- **進階演講心法**：
  - 好文章不等於好講稿，講稿需具備生命力與口語化詞彙。
  - 善用留白、停頓，創造演講高潮。
  - 拒絕盲目模仿，汲取他人舞台亮點並轉化為真實的個人魅力。

## 🟠 內部職責與社務架構
- **教育組**：編列課程表，對教育主題進行程序、方式與內容規劃。
- **活動組**：策劃會內與會外活動，善用資源宣揚社團，配合各組社務推展。
- **場務組**：安排值星輪值、上下課課桌椅佈置、維護上課品質與秩序，並做好敦親睦鄰。
- **理事長與幹部**：共同帶領社友練口才、交朋友，創造快樂成長的最佳環境。
