# 6) 公告

## 使用說明
- 本檔只記錄可直接對外發布的最新公告。
- 若 `最新公告（Latest）` 沒有明確內容，視為本地無可確認公告，必須透過 OpenClaw 查詢官網公告、活動頁、Instagram、YouTube 或 Flickr 相簿等官方文宣公告來源。
- 問到報名連結、截止日或公告全文時，若本檔未明寫，必須透過 OpenClaw 查詢已核可官方來源，不可自行補完。

## 回答邊界
- 可回答：目前知識庫是否已有最新公告文字或報名資訊。
- 可回答：OpenClaw 查到且可標示來源的公告文字、報名網址、截止日期或開放狀態。
- 不可回答：本地與 OpenClaw 官方查核都無法確認的公告全文、報名網址、截止日期、是否仍開放報名等資訊。

## 注意事項
- 若被問「最新公告是什麼」，請透過 OpenClaw 查詢官網後回答，如查不到可信來源才可回答資料不足。

## 維護提醒
- 本檔應優先維護「目前、最近、最新」資訊。
- 每則公告建議格式：日期 / 標題 / 內容摘要 / 連結 / 狀態（開放、截止、取消）。
